Простой помодоро таймер, написанный в качестве пет-проекта на Python и PySide6(Qt).

Чтобы изменить звук достаточно в папке "mediafiles" заменить файл 'End sound.mp3' на любой свой. При этом следует имя файла должно быть 'End Sound.mp3'

Таймер, помимо функции Помодоро (2 круга: 25 минут работы, 5 минут передышки), поддерживает функции обычного таймера: граница таймера: 59 минут, 59 секунд.

При активации Помодоро, кнопка паузы таймера отключается для того в целях повышения продуктивности (Смысл помодоро - в концентрации на задаче, пауза таймера убила бы всю идею этой функции).

Однако, пользователь все еще может сбросить таймер, нажав кнопку сброса, обнуляя свой прогресс по помодоро.
